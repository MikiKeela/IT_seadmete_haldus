# IT_seadmete_haldus

## Kirjeldus
Objektorienteeritud IT seadmete haldussüsteem käsurea liidesega (CLI).
Programm võimaldab lisada, kuvada, muuta ja kustutada seadmeid ning
salvestada ja laadida andmeid CSV ja JSON failidest.

## Lisaväli
Projektis on lisaväljana kasutatud `location` (seadme asukoht).

## Lubatud staatused
- available
- in_use
- broken

## Projekti struktuur
- models – Device klass
- services – DeviceManager äriloogika
- storage – CSV ja JSON salvestus
- ui – käsurea liides
- main.py – programmi käivitamine

## Käivitamine
Käivita fail `main.py`.
